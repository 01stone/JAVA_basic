## 42. 내부 클래스 (inner class)
> 클래스 안의 클래스
- 내부 클래스의 장점
  - 내부 클래스에서 외부 클래스의 멤버들을 쉽게 접근 가능
  - 코드의 복잡성을 줄일 수 있음(캡슐화)
